# FinTrack CLI

CLI-приложение для управления личными финансами (Java 17, Maven). Многопользовательская авторизация, категории, бюджеты, переводы между кошельками, экспорт/импорт и оповещения.

Быстрый старт:
1) mvn package
2) java -cp target/fintrack-cli-1.0.0.jar com.ddzhuk.fintrack.cli.Main

Команды: см. help внутри приложения.

Тесты: mvn test

Архитектура: core (модели/логика), infra (хранилище/авторизация), cli (интерфейс).
